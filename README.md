# Currency-converter
When the application starts, it accesses *Firebase Remote Config* and gets a list of cryptocurrencies. It is necessary to implement an activity that will display the rate of each cryptocurrency in real time. If you don't receive a response from Firebase with a list, you need to display a number of standard cryptocurrencies
For instance: 
-  Bitcoin
- Ethereum
- Litecoin
- Dogecoin

When you click on one of the cryptocurrencies, a chart should open with the change in the exchange rate of the selected currency for the last month/week
